## Getting Started

1.  **Configure Environment**: Copy `.env.example` to `.env` and fill in DB/API credentials.
2.  **Setup Database**: Run `npm run setup` to seed initial data.
3.  **Start Dev Server**: Run `npm run dev` to start the backend.
4.  **Access App**: Open `http://localhost:3000` (or configured port) in your browser.
