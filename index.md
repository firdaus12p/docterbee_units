# Directory Index - Docterbee Units (Root)

Docterbee Media & AI Server - Comprehensive health & wellness platform with Islamic perspective.

## HTML Pages (Frontend)

### Main Application
- **[index.html](./index.html)** - Landing page with featured products, journeys, and promotions
- **[about.html](./about.html)** - About Docterbee company and mission
- **[login.html](./login.html)** - User authentication login page
- **[register.html](./register.html)** - New user registration with card type selection
- **[profile.html](./profile.html)** - User profile dashboard with points and activities
- **[member-check.html](./member-check.html)** - Member status verification page

### E-Commerce
- **[store.html](./store.html)** - Product store with cart and checkout
- **[services.html](./services.html)** - Health service catalog and booking
- **[booking.html](./booking.html)** - Service booking form and confirmation

### Content & Education
- **[insight.html](./insight.html)** - Health articles and educational content
- **[article.html](./article.html)** - Single article reader with points-based access
- **[podcast.html](./podcast.html)** - Podcast episodes and audio content
- **[media.html](./media.html)** - Media gallery and promotional content
- **[journey.html](./journey.html)** - Gamified health journey (Zona Sunnah)

### Events
- **[events.html](./events.html)** - Upcoming health events listing
- **[event-registration.html](./event-registration.html)** - Event registration and ticket purchase

### AI Features
- **[ai-advisor.html](./ai-advisor.html)** - AI-powered health advisor using Gemini
- **[youtube-ai.html](./youtube-ai.html)** - YouTube video summarizer with AI
- **[docterbee-periksa-kesehatan.html](./docterbee-periksa-kesehatan.html)** - AI health check questionnaire

### Admin & Auth
- **[admin-dashboard.html](./admin-dashboard.html)** - Complete admin panel for content management
- **[reset-password.html](./reset-password.html)** - Password reset flow

### Error Pages
- **[404.html](./404.html)** - Custom 404 error page
